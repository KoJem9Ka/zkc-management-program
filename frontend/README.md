# zCred Frontend

## Development Environment

### Requirements

- [Node.js](https://nodejs.org)
- [pnpm](https://pnpm.io)

### Setup & Run dev environment

Install dependencies:
```bash
pnpm i
```

Run [zCred Store](../zcred-store/README.md) and configure its host in [.env](./.env) file.

Run dev server:
```bash
pnpm dev
```
